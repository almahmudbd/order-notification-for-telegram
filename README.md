## Order Notification for Telegram

Send WooCommerce order notifications to Telegram.

[![GitHub Downloads (all releases)](https://img.shields.io/github/downloads/almahmudbd/order-notification-for-telegram/total)](https://github.com/almahmudbd/order-notification-for-telegram/releases/latest)


## Description

This plugin allows you to receive WooCommerce order notifications directly on Telegram. You can customize the notification message and choose when to receive notifications.

## Features

- Instant order notifications
- Customizable message template
- Support for order status changes
- HTML formatting support
- Easy setup process

## Installation

1. Upload the plugin files to `/wp-content/plugins/order-notification-for-telegram`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce > Settings > Telegram Notifications to configure

## Configuration

1. Create a Telegram bot using @BotFather
2. Get your Chat ID from @userinfobot or @useridinfobot
3. Configure the plugin with your bot token and chat ID
4. Customize your notification message template
5. Test the setup using the test button
